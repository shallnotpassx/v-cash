# Phase 1: 股票模块功能补全

## 概述

在 Phase 0 股票列表/详情基础上，增加关键字搜索功能和定时自动同步任务，补全股票模块的核心交互。

## 动机

当前股票列表只能全量查看，无法按名称或代码快速定位目标股票。同时股票数据需要定期从数据源同步以保持最新，不能仅依赖手动刷新。

## 功能范围

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F1 | 关键字搜索 | 前端搜索框输入关键字（名称/代码），后端 LIKE 查询过滤 | P0 |
| F2 | 定时同步任务 | `@Scheduled` 每日自动从数据源同步股票列表 | P0 |

## 非功能需求

- 搜索支持大小写不敏感
- 搜索为服务端过滤（未来数据量大时不适用前端全量过滤）
- 定时任务同步失败不阻塞其他功能，仅记录日志

## 验收标准

- [ ] 前端搜索框输入关键字后，列表实时筛选显示匹配的股票
- [ ] 搜索支持按股票代码和股票名称匹配
- [ ] 服务端日志可观察到定时任务执行记录
- [ ] 定时任务可配置开关（默认开启）
